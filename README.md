# App Name 
***
### ○○市助成金チェッカー
***
# feature(特徴)
***
- 助成金（補助金）、手当のみに特化したアプリ。具体的にはピンポイントで自分が受給できる助成金をspeedyに検索し、当該助成金に関する情報の確認を可能にする。
- ユーザ間でお得な情報をシェアできるQ＆A機能付き。いいね、ベストアンサー機能を実装
※助成金等の内容は地方自治体によって異なるため、特定の市に限定した仕様となっています。
